1. 快速开始

1.1 老师注册与登录

  1. 打开网站 → 选择 注册 → 角色选 老师（Teacher）
  2. 填写邮箱、密码、显示名称；也可用 Google 登录
  3. 登录后左侧为功能菜单，仪表盘 为首页总览

1.2 添加学生（推荐：邮箱邀请）

  1. 进入 学生（Students） 或 班级（Classes） → 打开班级名册
  2. 点击 添加学生，填写 邮箱 + 姓名（可选年级、备注）
  3. 系统会 自动创建学生登录账号，初始密码：123456
  4. 告诉学生：打开同一网站 → 登录（不要注册）→ 邮箱 + 123456
  5. 学生首次登录后可在各页面看到自己的班级、课表、作业

备选： 在班级页分享 班级邀请码，学生自行注册后加入。

1.3 学生能做什么 / 不能做什么

可以 暂不支持或受限
看自己的课表、班级、作业 看不到其他学生隐私
提交作业、申请改课 未注册邀请生不能交作业
看自己的课时余额 —

2. 功能导览（老师）

2.1 仪表盘 Dashboard

  • 今日/近期课程、待办摘要
  • 快速了解「今天要上什么课」

2.2 日历 Calendar

  • 月视图 / 日视图 查看与排课
  • 老师：新建课程（选班级、日期、时间、标题、地点/会议链接）
  • 编辑 / 删除 已有课次
  • 记出勤：打开某节课 → 勾选出席学生
  • 页面下方有 AI 助手 卡片（测试版，见第 3 节）

2.3 班级 Classes

  • 新建班级：名称、颜色、计费方式等
  • 邀请码：学生可凭码加入
  • 名册：查看已加入学生；邮箱添加学生
  • 编辑 / 删除 班级

2.4 学生 Students

  • 全部学生 一览（跨班级）
  • 添加学生 到指定班级（邮箱 + 初始密码 123456）
  • 私有备注：仅老师可见，学生看不到
  • 点击学生可查看所属班级、备注

2.5 作业 Assignments

  • 按班级 布置作业（标题、说明、截止日期）
  • 查看 提交情况、批改与反馈
  • 学生提交后你会收到 通知

2.6 学费 / 课时 Tuition

  • 为学生 充值课时 或调整余额
  • 查看各学生余额与消费记录
  • 记出勤时可关联扣课时（若班级设为按课时计费）

2.7 通知 Notifications

  • 改课申请、作业提交、入班等系统提醒
  • 在 改课申请 中批准或拒绝学生请求

2.8 设置 Settings

  • 修改 显示名称
  • 头像（若已上传）

3. AI 助手（测试版 Beta）

入口： 左侧 AI 助手，或 日历 页底部卡片。

3.1 当前能做什么（只读）

用自然语言 查询 你自己的数据，例如：

  • 「这周有哪些课？」
  • 「Math 10 班有哪些学生？」
  • 「谁还没交作业？」
  • 「学费余额怎么样？」
  • 「有没有待审批的改课？」

AI 会先查系统真实数据再回答，支持 中文 / 英文。

3.2 使用注意（测试版）

  • 界面标有 「测试版」：功能与表述可能调整
  • 不能代替你改数据：排课、出勤、充值等请在对应页面操作
  • 重要决定前请在应用里 再核对 班级名、日期、学生姓名
  • 仅 老师账号 可用；学生端暂无 AI

3.3 即将推出（收集你的反馈后开发）

  • 用说话方式 排课 / 改课 → 弹出确认卡 → 你点确认后才执行
  • 记出勤、充课时、布置作业、写备注等（均需确认）
  • 可选：从 Excel/PDF 导入学生名单

4. 常见问题

Q：学生说登录不了？
A：确认用 登录 而非注册；密码是否为 123456。若仍失败，在 学生 页用同一邮箱 再添加一次 到班级，系统会修复账号状态。

Q：邀请的学生显示「待加入」？
A：多为旧邀请记录。请 重新添加 该邮箱，或让学生用 123456 登录。

Q：AI 没反应？
A：需管理员在服务器配置 AI 密钥；若提示未配置，请联系平台维护者。

Q：邮件通知收不到？
A：当前版本以 站内通知 为主；邮件功能仍在完善。

5. 测试时请你重点体验

请老师在内测中重点试这些流程，并反馈是否顺手：

  1. 建班 → 邮箱添加 1～2 名测试学生 → 学生用 123456 登录
  2. 日历排课 2～3 节 → 记出勤 → 看仪表盘/日历是否一致
  3. 布置一条作业 → 学生提交 → 老师批改
  4. 学费充值 → 出勤后看余额变化
  5. 学生发起改课申请 → 老师批准/拒绝
  6. AI 助手：问课表、学生、作业各 1 次，看回答是否准确

反馈方式： 直接告诉产品/开发负责人（问题页面、操作步骤、截图更佳）。
